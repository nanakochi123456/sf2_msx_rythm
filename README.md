sf2_msx_rythm
=============
MSXエミュレータで採取したリズムをSoundFontにしてみたものなの。

(MSX OPLL&PSG DRUM's SoundFont)

ツールは、Polyphoneを使ってなの。

注意：
	Bank128に作ることなの
